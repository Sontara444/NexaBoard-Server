# NexaBoard Server

Simple Task Management API built using **Node.js**, **Express**, and **MongoDB**.

## Setup

```bash
npm install
npm run dev
```



## Scripts

```bash
npm run dev   
```

## Project Structure

```
src/
  controllers/
  middleware/
  models/
  routes/
  utils/
server.js
```

## Features

* User authentication
* CRUD tasks
* JWT protected routes
